# What is Cat?

**A quick bootstrap for a human to understand a cat**

Short answer: **A cat is a highly optimized chaos-powered biological machine.**

## About

This website provides essential knowledge about how cats work, why they do what they do, and how they decided humans were acceptable. The content is organized into three main topics:

- **How Cats Work**: Understanding the fundamental systems and history of cats
- **Behavior**: Exploring why cats do what they do
- **Communication**: Understanding how cats communicate with humans

## Visit the Site

The site is deployed at: **https://whatiscat.outright.io/**

## Local Development

This site is built with [Hugo](https://gohugo.io/), a fast static site generator.

### Prerequisites

- Hugo Extended v0.139.0 or later

### Running locally

```bash
# Build the site
hugo

# Start the development server
hugo server

# Build for production
hugo --minify
```

The site will be available at `http://localhost:1313/whatiscat/`

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch via GitHub Actions.
